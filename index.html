Variables

Variables are used to store data in memory

Creating and initializing vars will allocate memory to store some data or value in memory and provide a reference to the location of data in memory

Declare variables with let or var keywords:

  1. let myVariable = 'some string'
  2. var badVariable = 'this is the old way'
